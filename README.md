# gb-rs

A GameBoy emulator, written in Rust.

### Features
- Supports MBC1/MBC2/MBC3/MBC5 DMG Titles
- Sprite & BG Rendering
- Cycle-Accurate CPU
- Cross-Platform
- Cross-Graphics API (Metal, Vulkan, OpenGL, D3D12, WebGPU)

### Not Complete
- CGB Support
- Sound
- Input
- UI/Config Settings
